# Back-end-Development-Capstone

This capstone project demonstrates a complete backend development application using Python and Flask. The project implements REST API services with CRUD functionality, automated testing, and API endpoint validation.

## Features

- Health check endpoint
- CRUD operations for resources
- RESTful API implementation
- Automated testing with Pytest
- JSON request and response handling
- Error handling and validation

## API Endpoints

### Health Endpoint
- GET /health

### Resource Endpoints
- GET /items
- GET /items/<id>
- POST /items
- PUT /items/<id>
- DELETE /items/<id>

## Technologies Used

- Python
- Flask
- Pytest
- GitHub
- Docker
